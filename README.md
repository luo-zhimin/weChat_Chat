# weChat_Chat
基于SpringBoot+Socket开发实现微信(公众号)扫码登陆 实时通信

|     依赖     |  说明   |
|:----------:|:-----:|
| SpringBoot | MVC框架 |
|  Mybatis   | ORM框架 |
|   Mysql    |  数据库  |
|   Redis    |  缓存   |
|   Hutool   | Token |
| WebSocket  | 实时通信  |

- 公众号扫码登录完成
